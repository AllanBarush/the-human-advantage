## Deployment

This project is designed for **GitHub Pages** deployment:

1. Push this repository to `allanbarush/the-human-advantage`
2. Enable GitHub Pages in Settings → Pages → Source: `main` branch, root `/`
3. The landing page will be live at `https://allanbarush.github.io/the-human-advantage/`

## Email Templates

### Campaign Email (`emails/email-campaign.html`)
- Compatible with **iContact** and major email service providers
- Uses table-based layout for maximum email client compatibility
- Includes MSO conditional comments for Outlook rendering
- Responsive design with mobile breakpoints
- Preheader text for inbox preview optimization

### Confirmation Email (`emails/email-confirmation.html`)
- Auto-response template for registration confirmations
- Program summary card with all key details
- Next steps timeline
- Contact information for follow-up
